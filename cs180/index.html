Project 0

Becoming Friends with Your Camera

Take a fresh look at how cameras work by exploring how different focal lengths and viewpoints affect image formation and distortion.

Project 1

Images of the Russian Empire

Reconstruct photographs from historic glass-plate negatives that contain three images of the same scene captured through blue, green, and red filters. The goal is to align the three channels so that corresponding features match, producing a coherent full-color image. I first implement single-scale alignment, testing pixel shifts to find the best overlap, and then improve efficiency and accuracy with multi-scale pyramid alignment, from coarse to fine resolution.

Project 2

Fun with Filters and Frequencies!

Explore fundamental 2D image processing by operating in both the spatial and frequency domains. Part 1 focuses on mastering 2D convolutions and using them with various filters to achieve effects like edge detection and Gaussian blurring. Part 2 applies filtering principles on advanced image manipulation, including image sharpening, creating hybrid images by blending high- and low-frequency components, and achieving seamless image blending using multiresolution techniques.

Project 3

[Auto] Stitching Photo Mosaics

Create large composite images (panoramas) by stitching multiple photographs together. Part A requires manually selecting correspondent points between overlapping images, and calculating the homography matrix to warp and composite them into a seamless mosaic. Part B automates this process to implement a technique for automatically finding feature correspondences to eliminate the need for manual point selection, thereby enabling fully automatic panorama generation.
